Пошук
• В'їзди на паркування знаходяться пошуком
• Поліпшено зіставлення вулиць та релевантність результатів
• Пошук продуктових магазинів та товарів для дому

Стилізація картки
• Нові або оновлені іконки для заводу, виноробні, книжкової крамниці, бібліотеки, станції ремонту велосипедів, прокату велосипедів, велопарковки, сауни, лотереї, букмекерської контори, масажу, шиття, книг, підземного паркінгу, місця для паркування інвалідів, воріт, торгівлі пакетами для екскрементів, оптового магазину, місць для відпочинку на свіжому повітрі, ремесла, млина, кейтерингу, вентиляції та кондиціювання повітря, майстерні з виготовлення ключів та слюсарних робіт
• Додані автоінспекція, громадська лазня, велобар'єр, водні шляхи (стік, канава, рів, стічні води), акваріум, пункт збору, зал атракціонів, мормонські церкви, центр відвідувачів, промислова шахта, історичні об'єкти (гармата, якір, камінь, літак, танк, шахта, уламки, локомотив)
• Додано рендеринг мостів та тунелів для велодоріжок, пішохідних доріжок, стежок, доріжок, мостових, сходів
• Рендеринг рову (бар'єру) відрізняється від рендерингу рову (водного шляху)
• Зменшено пріоритети назв річок (та інших малозначимих ліній) у режимі навігації
• Налаштовані пріоритети назв доріг та дорожніх щитів
• Виправлено відображення іконок будівель паркування
• Зменшено видимість іконок дрібних та приватних паркувань, видалено заливання площі для підземних парковок
• Зменшено видимість пожежних гідрантів
• Номери будинків відображаються над іконками та написами основних POI, а також не відображаються номери будинків для будівель, які надто малі для поточного масштабу
• Не відображати піктограму зірки для квадратів
• Уточнити шрифт номерів будинків та пріоритет малювання
• Додані лінії пішохідних, мостових та велосипедних доріжок на детальному масштабі (z18+) у режимі навігації
• Приховані пішохідні тунелі в режимі навігації
