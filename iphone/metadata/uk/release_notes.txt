Новий стиль для орієнтування на природі:
• Пристосований для тих, кто проводить час за містом, ходить в походи, їздить по пересіченій місцевості на велосипедах чи позашляховиках
• Акцентує увагу на пішохідних стежках, кручах, залізничних дорогах, лініях електропередач, річках та струмочках, джерелах води, вершинах, гірських перевалах та стежках, водопадах, маяках, печерах, оглядових майданчиках, кемпінгах, хатинках, укриттях, місцях для пікніку, воротах, парканах, тощо
• Щоб увімкнути новий стиль введіть `?olight` в поле пошуку та поверніться до мапи. Введить `?light` щоб повернути стандартний стиль
• Використовуйте `?odark` та `?dark` для темного режиму
• Закликаємо розробників реализувати більш зручний UI для перемикання природного стилю!
