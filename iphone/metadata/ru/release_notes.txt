• Обычное нажатие на карту теперь позволяет выбрать любую точку, даже если карта пуста
• Длительное нажатие на карту переключает полноэкранный режим

• Данные OpenStreetMap по состоянию на 2 июля
• Выбранное место теперь показывает полный список типов (категорий) объектов в данной точке
• Случайно удаленную метку на карте теперь можно сразу же восстановить (кнопка "Сохранить" меняется на "Восстановить")
• KML, KMZ, GPX файлы можно импортировать с помощью кнопки Импорт в диалоге "Метки и треки" для загрузки файлов KML, KMZ, GPX (метки и треки также можно импортировать, "поделившись" ими из любого другого приложения, а затем выбрав "Открыть с помощью Organic Maps")
• Исправлен импорт KML, KMZ, GPX через приложение "Файлы" при первом запуске приложения
• Добавлена кнопка проверки голосовых инструкций (TTS) в настройках навигации
• Теперь можно включить запись логов в настройках, чтобы помочь нам в исправлении ошибок
• Ссылки на соцсети открываются в установленных приложениях
• Удалена неработающая кнопка пробок из CarPlay
• Круглые кнопки в редакторе времени заменены на закругленные прямоугольники
